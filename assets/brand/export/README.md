## Usage Notes

- Use the horizontal logo for presentations, documentation, and top-left app branding.
- Use the standalone mark for favicon, compact sidebar, avatars, and icon-only contexts.
- Do not use the old tagline in the product UI.
- Keep clear space around the logo at least equal to the width of one mark segment.
- Prefer light-mode logo on white product UI and dark-mode logo on dark/presentation backgrounds.
