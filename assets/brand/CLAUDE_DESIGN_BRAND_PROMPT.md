# Claude Design Prompt: MonoPulse Branding

Design a polished brand system for **MonoPulse**, a B2B backoffice product for iGaming operators. MonoPulse helps CRM, retention, and casino managers configure gamified campaigns, loyalty programs, rewards, platform/CRM segment targeting, trigger logic, risk controls, approvals, analytics, and money/liability monitoring.

## Brand Personality

MonoPulse should feel:

- precise, operational, and trustworthy
- modern fintech/admin rather than playful casino
- intelligent and data-aware
- calm under pressure
- built for repeated daily use by operators
- premium, technical, and commercially focused

Avoid:

- childish gamification visuals
- casino clichés such as neon chips, cards, dice-heavy styling, jackpot explosions
- loud gradients or decorative blobs
- marketing landing-page energy
- overly soft SaaS illustration styles

## Visual Direction

The product is a serious control room for gamification, rewards, campaign logic, and financial exposure. The brand should balance:

- **Control**: governance, permissions, audit, safety
- **Pulse**: real-time events, live triggers, dynamic audiences
- **Growth**: retention, loyalty, campaign performance
- **Precision**: money, risk, liability, backend logic

Use compact, professional UI language. This is an operator console, not a consumer app.

## Existing Logo

The current mark is a geometric double-pulse / angular `M` symbol in mint green, paired with a `MonoPulse` wordmark.

Use the existing logo as source direction:

- primary logo: horizontal mark + MonoPulse wordmark
- compact logo: standalone mark
- no tagline in the product UI
- preserve the strong geometric mark idea

## Typography

Use a clean modern sans-serif suitable for dense dashboard UI. Prioritize legibility, numeric clarity, and enterprise seriousness.

Recommended direction:

- UI font: Inter, Geist, or similar
- Numbers: tabular numerals / mono-style treatment for financial values
- Headings should be restrained, not oversized
- Avoid playful rounded display fonts

## UI / Brand System Needs

Create a brand kit that includes:

- primary logo usage on light and dark backgrounds
- standalone mark usage
- favicon/app icon guidance
- color tokens
- typography direction
- button and status color examples
- dashboard card examples
- data visualization style, including subtle sparklines and money/risk states
- examples for product surfaces: Dashboard, Campaign Builder, Rewards, Risk & Compliance

## Tone of Voice

Voice should be concise and operational.

Use phrases like:

- Dashboard
- Commercial control view
- Platform segment
- Trigger logic
- Projected reward cost
- Open liability
- Approval required
- Last synced
- Dynamic audience

Avoid phrases like:

- magic
- fun rewards
- spin to win excitement
- money dashboard
- finance priority
- childish gamification language

Please produce a professional brand direction for MonoPulse, including:

1. Logo usage recommendations
2. Color palette refinement
3. Typography recommendation
4. UI component style guidance
5. Dashboard visualization style
6. Do/don't examples
7. A compact brand summary suitable for handoff to design and engineering
