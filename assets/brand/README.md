# MonoPulse Brand Assets

- `monopulse-logo-source.png` is the original Higgs reference image.
- `monopulse-logogram.png` is the standalone app mark used by the prototype UI.
- `/public/monopulse-logogram.png` is the small browser icon generated from the app mark.

The app renders the product name as live text beside the logogram so the tagline is never baked into the interface.
