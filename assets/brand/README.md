# MonoPulse Brand Assets

- `monopulse-logo.svg` is the primary logo and source of truth for the prototype UI.
- `monopulse-logo-light.svg` is the light-mode logo with dark wordmark text.
- `monopulse-mark.svg` is the standalone app mark used for compact sidebar and favicon contexts.
- `monopulse-mark-light.svg` is the light-mode standalone mark.
- `/public/monopulse-mark.svg` is the browser favicon copy.
